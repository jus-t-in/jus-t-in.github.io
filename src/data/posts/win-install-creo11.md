# Windows 安装 Creo11
虚拟机内安装 Creo11。。。
详见：https://mp.weixin.qq.com/s/wnQRP-VYTvfHpAe1wrU7Aw

## 一、下载 Creo11
链接：https://pan.xunlei.com/s/VOq4zUMvZjAruJ1M_wYEJ3dOA1
提取码：xr26

## 二、安装

1.解压文件

2.复制【Crack】文件夹中的【Lincense】，粘贴到c盘根目录

3.打开c盘的【Lincense】，运行【FillLicense】

4.回车后，【Lincnese】文件夹下会生成一个【PTC_D_SSQ.dat】文件

5.①鼠标右键桌面【此电脑】；②选择【属性】；③点击【高级系统设置】

6.点击【高级】，点击【环境变量】

7.①在系统变量下方点击【新建】；②输入变量名【PTC_D_LICENSE_FILE】和变量值【C:\Lincese\PTC_D_SSQ.dat】；③点击【确定】；④点击【确定】
![alt text](images/win-install-creo11/image1.png)

8.打开安装包解压后的【Creo11.0(64bit)】文件夹，双击打开【Setup】文件夹

9.鼠标右键【Setup】，选择【以管理员身份运行】

10.【下一步】，同意许可【下一步】，把生成的【PTC_D_SSQ.dat】文件拖动到软件界面内，【下一步】
![alt text](images/win-install-creo11/image2.png)

11.修改安装路径，【下一步】，【安装】，点击【完成】

12.打开安装包解压后的【Crack】文件夹，鼠标右键【Creo11.0.0.0】选择【复制】

13.打开【Creo】快捷方式的【文件所在的位置】，点击路径地址中的【PTC】，把刚才复制的【Creo11.0.0.0】粘贴到此处

14.打开安装包解压后的【Creo11.0(64bit)】文件夹，双击【SolidSQUADLoaderEnabler】，修改注册表

15.完成安装，打开软件
